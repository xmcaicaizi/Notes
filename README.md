## 1、梯度下降
###  Part 1: 梯度下降的主要原理  
1.确定一个小目标--预测函数  
        ```
        学习算法  
        发现规律  
        改进模型  
        做出预测
        ```  
2.找到差距--代价函数  
```
例如均方误差:  
e = (y - w * x)²  
展开后：  
e = x² * w² - 2(x * y) * w + y²  

多个e1 = x1² * w1² - 2(x1 * y1) * w1 + y1²  
e2 = x2² * w2² - 2(x2 * y2) * w2 + y2²  
e3 = x3² * w3² - 2(x3 * y3) * w3 + y3²  
...  
en = xn² * wn² - 2(xn * yn) * wn + yn²  
```
